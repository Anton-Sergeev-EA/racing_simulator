## Сборка и запуск.

### Требования.
- C++20
- CMake 3.20+

### Сборка.

mkdir build && cd build
cmake ..
cmake --build . --config Release

### Запуск:
cd bin
./RaceSimulator          # Linux/macOS
RaceSimulator.exe        # Windows

## Структура проекта:

racing_simulator/

Реализованные требования:
Функциональность.
- Выбор типа гонки (наземная/воздушная/смешанная).
- Ввод дистанции.
- Регистрация ТС (не более одного каждого вида).
- Проверка совместимости ТС с типом гонки.
- Минимум 2 ТС для старта.
- Расчёт времени с учётом остановок (наземные) и коэффициентов (воздушные).
- Вывод результатов по возрастанию времени.
- Возможность провести ещё одну гонку.

Архитектура.
- Иерархии классов (Vehicle и Race).
- Отдельная динамическая библиотека (DLL/SO).
- В библиотеке нет #include <iostream>.
- В EXE только ввод/вывод, без логики расчёта.
- CMake.
